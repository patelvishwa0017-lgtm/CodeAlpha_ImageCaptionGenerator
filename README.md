# CodeAlpha Image Caption Generator

## Project Overview

This project uses Artificial Intelligence to automatically generate captions for images using a pre-trained Transformer model. The system takes an input image and produces a descriptive caption.

## Technologies Used

* Python
* Transformers
* PyTorch
* Pillow
* Google Colab

## Features

* Upload an image
* Generate captions automatically
* Save generated captions to a CSV file
* Uses a pre-trained AI model for image understanding

## Files Included

* CodeAlpha_ImageCaptionGenerator.ipynb
* image_caption.csv
* README.md

## How to Run

1. Open the notebook in Google Colab.
2. Install the required libraries.
3. Upload an image.
4. Run the notebook cells.
5. Generate and save the image caption.

## Example Output

Input: Dog Image

Generated Caption:
"A dog running through the grass."
